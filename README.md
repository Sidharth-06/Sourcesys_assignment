# Sourcesys Assignment

This branch adds three small Python command-line programs:

- `python_programs/calculator.py` - basic arithmetic for two numbers
- `python_programs/palindrome.py` - palindrome detection with normalization
- `python_programs/word_stats.py` - counts lines, words, and characters

## Examples

```bash
python python_programs/calculator.py 7 + 5
python python_programs/palindrome.py "A man, a plan, a canal: Panama"
python python_programs/word_stats.py "hello world"
```
